# `void SetValue()`

### Sets the value contained within the `Node<T>`.

## Exceptions:
- None.

## Example:

```cs
Node<int> a = new Node<int>(5);
Console.WriteLine(a.GetValue()); // 5
a.SetValue(6);
Console.WriteLine(a.GetValue()); // 6
```
